Methodology

National counts come from the Nager.Date “Countries” table. Regional counts and per-province breakdown are computed from the 2025 PublicHolidays feed (unique dates per region). The page makes one local request to a prebuilt JSON file; clicking a country fetches cached details via a serverless proxy.

About

Holiday Atlas is a portfolio demo mixing secure data handling and fast client UX.

Data freshness

Static totals are rebuilt offline. The details API caches responses for 7 days.
